# شبیه‌سازی مطب پزشک

پروژه شبیه‌سازی مطب پزشک با استفاده از Next.js و Tailwind CSS v4

## ویژگی‌ها

- شبیه‌سازی کامل فرآیند مطب پزشک
- مدیریت صف‌های مختلف (ورودی، پذیرش، پرداخت، انتظار، ویزیت)
- کنترل تعداد باجه‌های پذیرش و دکترها
- نمایش آمار و نمودارهای آماری
- رابط کاربری زیبا با Tailwind CSS v4

## نصب و راه‌اندازی

1. نصب وابستگی‌ها:
```bash
npm install
```

2. اجرای پروژه در حالت توسعه:
```bash
npm run dev
```

3. باز کردن مرورگر در آدرس:
```
http://localhost:3000
```

## ساختار پروژه

```
├── app/                    # صفحات و layout اصلی
│   ├── layout.tsx         # Layout اصلی
│   ├── page.tsx           # صفحه اصلی
│   └── globals.css        # استایل‌های全局
├── components/            # کامپوننت‌های React
│   ├── ClinicHeader.tsx   # هدر مطب
│   ├── PatientCard.tsx    # کارت بیمار
│   ├── EntranceArea.tsx   # بخش ورودی
│   ├── ReceptionArea.tsx  # بخش پذیرش
│   ├── PaymentArea.tsx    # بخش صندوق
│   ├── WaitingArea.tsx    # سالن انتظار
│   ├── DoctorOffice.tsx   # اتاق دکتر
│   ├── ControlPanel.tsx   # پنل کنترل
│   └── StatsChart.tsx     # نمودار آمار
├── hooks/                 # Custom hooks
│   └── useClinicSimulation.ts  # هوک اصلی شبیه‌سازی
└── types/                 # TypeScript types
    └── patient.ts         # انواع داده‌های بیمار
```

## استفاده

1. **افزودن بیمار**: با کلیک روی دکمه "افزودن بیمار جدید" می‌توانید بیمار جدید به سیستم اضافه کنید.

2. **شروع شبیه‌سازی**: با کلیک روی دکمه "شروع شبیه‌سازی" فرآیند شبیه‌سازی شروع می‌شود.

3. **تنظیم تعداد باجه‌ها**: می‌توانید تعداد باجه‌های پذیرش (1-5) و تعداد دکترها (1-3) را تنظیم کنید.

4. **کنترل سرعت**: با استفاده از اسلایدر می‌توانید سرعت شبیه‌سازی را تنظیم کنید (0.5x تا 2x).

## تکنولوژی‌ها

- **Next.js 14**: فریمورک React
- **TypeScript**: برای type safety
- **Tailwind CSS v4**: برای استایل‌دهی
- **Chart.js**: برای نمایش نمودارها
- **React Hooks**: برای مدیریت state

## ساخت و تولید

برای ساخت نسخه production:

```bash
npm run build
npm start
```
